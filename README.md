# Symmetric Square Grid

A programming challenge about arranging square tiles into a rectangular grid with symmetric empty cell placement.

## 📋 Problem Description

Given `n` identical square tiles, arrange them into a rectangular grid following specific rules:
- Minimize the difference between rows and columns (make it as square as possible)
- Rows must be ≥ columns
- Empty cells are placed symmetrically according to a specific pattern

## 🎯 Challenge

The tricky part isn't the grid dimensions calculation — it's figuring out the exact pattern for placing empty cells. The rule is deterministic but non-obvious. Study the examples carefully!
